# 한달인턴 프론트엔드 개발 온보딩 과제 (React)

기본적인 JWT 기반의 인증/인가가 구현되어있고 프로필 수정이 가능한 간단한 웹 사이트

## 페이지 구성

- 메인 페이지
  - JSON Placeholder 테스트
  - 마이 페이지 이동
  - 로그아웃 기능
  - 로그인 여부 상관 없이 접속 가능
- 로그인 페이지
  - 로그인 시 접속 불가
- 회원가입 페이지
  - 로그인 시 접속 불가
- 마이 페이지
  - 프로필 변경 가능
  - 로그인 시에만 접속 가능
